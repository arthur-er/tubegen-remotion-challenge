## Setup and render instructions

1. Clone the repository
2. `$ cd starter`
3. `$ pnpm install`
4. `$ pnpm render`

## Approach to design and animation and tradeoffs

This was the biggest tradeoff, I used AI to generate various design prototypes and the one I liked the color pallete the most I gone forward.

For animation I follow Emil Kowalski tips on animation, and try to keep everything smooth, but as time the time was very small, there's still some staggering and some animations aren't that good. 

## What you'd improve given more time

I would create a design of the app first, then use the app shell in the video, making everything look consistent. I would also pay more attention to the animation timings to make everything synced and smooth
